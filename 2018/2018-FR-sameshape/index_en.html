It's computer science !

In computer science, pattern recognition and transformation plays a major role : these rules are called rewriting rules !

They are very useful to enable computers to recognize programming languages: it is called parsing.
Indeed, a language syntax can be defined with a set of rewriting rules that we call grammar.

They can also be used to model many phenomenon: for instance, chemical or biological transformations.

Actually, everything that can be computed with a computer can also be computed with only rewriting rules.
There exists a theorical programming language that is only based on rewriting : the lambda-calculus.
